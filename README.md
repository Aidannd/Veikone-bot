 # VEIKONE - BOT

![](./src/veiko.gif)

## INSTALLATION

#### REQUIRE
1. [nodejs](https://nodejs.org/en/download) 18x/20+

##### CLONE REPOSITORIES
```Alpine Abuild
git clone https://github.com/Aidannd/Veikone-bot
cd Veikone-bot
```

#####  HOW TO INSTALL DEPENDENCIES
```Alpine Abuild
npm i
```

#####  START BOT
```Alpine Abuild
node index.js
```
